Ejercicio 1: Procesamiento de secuencias

Desarrollo

Lo primero que hicimos fue determinar un gen a utilizar, en este caso fue el HLA-DQA1 que se encuentra relacionado por ejemplo con la celiaquía.

Luego, en la base de datos del NCBI (National Center for Biotechnology Information) consultamos las secuencias de referencia, seleccionamos uno de los mRNA (NM_002122.3) y descargamos el transcripto del gen en formato GenBank.

A continuación realizamos el script que hace lo siguiente:

Utiliza el archivo descargado (o el que se le envie por línea de comandos), traduce la secuencia teniendo en cuenta el marco de lectura correcto (verificando previamente que sea válida), y finalmente genera un nuevo archivo en formato fasta con el resultado.

Funcionamiento

Desde línea de comandos ingresar:

> perl Ex1.pm [--f archivo_GenBank]

  • Parámetros

    • --f archivo_fasta: Archivo de tipo genbank utilizado como input. (Default: 'sequence.gb') [Opcional]

  • Input

    • Archivo en formato GenBank.

  • Output

    • Archivo en formato Fasta.

    Si el script logra ejecutarse correctamente, como resultado se obtendrá un nuevo archivo con el mismo nombre que el utilizado como Input, pero con extensión "*.fas".

Ejemplos

  • Ejemplo 1

    • > perl Ex1.pm

    - Al no indicarse ningún archivo específico, el programa buscará automaticamente el archivo "sequence.gb" y lo utilizará. Como resultado se generará el archivo "sequence.fas".

  • Ejemplo 2

    • > perl Ex1.pm --f insulina.gb

    - El programa buscará (en la misma carpeta) un archivo llamado "insulina.gb" y lo utilizará.

    Como resultado se generará el archivo "insulina.fas".
