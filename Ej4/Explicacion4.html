Ejercicio 4: Blast output

Desarrollo

Lo que hicimos fue utilizar el archivo blast obtenido del ejercicio 2, como input para este ejercicio.

El script se encarga de detectar, de dichos resultados, aquellos hits en cuya descripción se encuentre el patrón que sea indicado como parámetro.

Además, en base al accession code de estos hits (que coincidieron con el pattern), se encarga de obtener sus secuencias completas y crear un archivo en formato FASTA para cada uno de ellos.

Funcionamiento

Desde línea de comandos ingresar:

> perl Ex4.pm [--f archivo_blast] [--p pattern] [--cs]

  • Parámetros

    • --f: Archivo blast utilizado como input. (Default: 'sequence.blast').

    --p: Pattern a detectar

    --cs: Indica si la búsqueda será case-sensitive, es decir, teniendo en cuenta mayúsculas y minúsculas. (Default: false).

    * El único parámetro obligatorio es el patrón

  • Input

    • Archivo en formato Blast.

  • Output

    • Archivos en formato Fasta.

    Si el script logra ejecutarse correctamente, como resultado se obtendrá, para cada coincidencia detectada, un archivo Fasta con el accession code como nombre del mismo. (AccessionCode.fas)

Ejemplos

  • Ejemplo 1

    • > perl Ex4.pm --p Tapasin

    * Al no indicarse ningún archivo específico, el programa buscará y utilizará automaticamente el archivo "sequence.blast". Al no indicarse el flag --cs, no se tendrán en cuenta las mayúsculas ni minúsculas. Como resultado se generarán tantos archivos como coincidencias encontradas.

    En este caso existen 3 coincidencias: Q9BX59.2, Q5R8H1.2, O73895.2, cuyos respectivos archivos serán:

    • Q9BX59.2.fas
    • Q5R8H1.2.fas
    • O73895.2.fas

  • Ejemplo 2

    • > perl Ex4.pm --p TaPAsIn --cs

    * Nuevamente el programa utilizará como input el archivo "sequence.blast". Pero en este caso, la búsqueda será case-sensitive. Al no encontrar coincidencias, no generará ningún archivo.

  • Ejemplo 3

    • > perl Ex4.pm --f otroArchivo.blast --p Arabidopsis

    * El programa utilizará como input el archivo "otroArchivo.blast", y detecctará aquellos hits en cuya descripción se encuentre el patrón "Arabidopsis", sin hacer distinción entre mayúsculas y minúsculas.
